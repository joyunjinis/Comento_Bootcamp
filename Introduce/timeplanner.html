Team Planner

Team Planner는 일정 조율 프로그램입니다.
초중고 및 대학생들이 평소 조별 과제가 있을때,
쉽고 편리하고 효율적이게 일정을 조율하고
파일 공유 및 여러 플랫폼을 사용하지 않고,
한 플랫폼에서 모든 걸 한번에 사용할수 있도록
지원해주는 프로그램 입니다.

주요 기능으로는 REST API 서버 구현, 회원가입 및 로그인(인터페이스 포함), 일정 관리 시스템, 사용자 초대, 실시간 채팅, 알림,
관리자 권한 설정, 자료 공유, 검색 등이 있습니다.

주요 정보

학교 팀 프로젝트

참여인원: 3人(Full Stack)

담당 기능

  • 자체 회원가입 및 로그인 구현
  • 소셜 로그인 API 구현(KAKAO, GOOGLE)
  • 파일 업로드 및 다운로드 구현

스킬


트러블 슈팅

  • 소셜 로그인 API 구현 중 각각의 코드 충돌
  • 파일 다운로드 과정에서 파일명(한글) 깨짐 발생
  • [전체] 코드 합치는 과정에서 충돌
